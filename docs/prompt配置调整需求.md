调整“Prompt 配置”的整体配置方法，目标是标准化输出让其可以匹配模板字段的结构并写入。

0.在“Anki 连接”tab 中选择了模板后，根据模板字段，在“Prompt 配置”页面显示对应的字段配置区域。
区域顶部是所有字段的展示区域，每个字段有选中和未选中的状态样式。选中的字段会在下方显示该字段的配置行。
每个被选中的字段的配置行中包括字段名称（文本），字段内容（多行文本），样例。其中字段内容为必填；
如果取消展示区域中字段的顶部选择，该字段行也会从下方取消。

1.在字段配置区域下方，显示一个 prompt 预览区域，该区域可进行编辑，此处 prompt 的内容是最终供 AI 解析用的 prompt。
prompt 预览区域的上部，在标题的右侧放置一个“prompt 重置”文字按钮，点击后根据被选择的字段以及设置内容，生成一个默认的完整 prompt。
默认的完整 prompt 会分为两部分，分别是字段返回内容定义和输出格式定义。

        （1）字段返回内容定义：根据上方选择的字段名称，字段内容，样例（若有），自动组合，比如：
            正面：为输入文本本身
            背面：
                请按顺序完成以下所有步骤后输出的内容：
                步骤 1: 生成YouGlish链接
                根据输入的日语单词，生成对应的YouGlish日语发音URL。
                    格式为：`https://youglish.com/pronounce/[URL编码后的日语单词]/japanese`
                请直接输出这个URL。

                步骤 2: 提供详细单词信息
                在生成URL后，请接着严格按照以下任务要求，提供关于该单词的详细信息：

                任务1 (Task 1): 提供该单词的 纯假名发音 (不含罗马音)、中文释义、详细解释。
                    请确保这三个信息项（及其标签）各占一行
                任务2 (Task 2): 提供该单词的 日语词源（如果已知或普遍）、日语衍生用法、日语相似用法（如有）。
                    请确保这三个信息项（及其标签，如果存在）各占一行，并遵循下方模板中的格式。

                输出格式: 请将任务1、2的结果严格按照以下格式排版输出：
                    单词释义
                    假名发音：[此处插入纯假名发音]
                    中文释义：[此处插入中文释义]
                    详细解释：[此处插入详细解释]

                    扩展
                    词源：[此处插入日语词源信息，如无则注明“暂无”或省略该行]
                    衍生用法：[此处插入日语衍生用法信息，如无则注明“暂无”或省略该行]
                    相似用法：[此处插入日语相似用法信息，如无则注明“暂无”或省略该行]

            例句：使用该单词构造2个不同的日语例句，并为每个例句提供中文翻译。

        （2）输出格式定义：需要明确返回的数据格式为json，参数名必须为字段名，比如：

            请按以下JSON结构输出信息，保持字段名准确：
            输出 JSON 结构示例：
            {
              "正面":"上文定义的正面内容",
              "背面":"上文定义的背面内容",
              "例句":"上文定义的例句内容"
            }
